# 💊 KAIN-Aptheka

Учебный Full-stack проект системы заказа лекарств.

# 🚀 Технологии:
- **Frontend:** HTML5, CSS, JavaScript.
- **Backend:** Node.js, Express.
- **Database:** PostgreSQL.

# 🛠 Инструкция по запуску:
1. Клонируйте репозиторий.
2. Выполните `npm install`.
3. Создайте базу данных в PostgreSQL и выполните SQL-скрипт из файла `database.sql`.
4. Создайте файл `.env` на основе примера и введите свои данные БД.
5. Запустите сервер командой `node index.js`.
